# Eggode: Mobile (Яйцеоды: Mobile) - фанатский проект по "Яйцеодам 2"

Проект ппредставляет из себя порт одноимённой игры на мобильные устройства Android (в будущем возможно и IOS)

---

## Дисклеймер: О качестве кода

Проект не меняет оригинальную структуру. Код написан посредством вайбкодинга и прочего, просьба не прикапываться к коду.
